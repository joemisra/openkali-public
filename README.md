# Embedded C++ Synthesizer Project

## Build Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/joemisra/openkali-public.git
   cd openkali-public
   ```
2. Install the required dependencies:
   ```bash
   sudo apt-get install <dependencies>
   ```
3. Build the project:
   ```bash
   make
   ```
